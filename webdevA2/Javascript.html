Javascript

History of Javascript and What is Javascript?

Javascript was invented by Brendan Eich in 1995 for Netscape 2, it became the ECMA-262 standard in 1997. When Netscape handed Javascript over to ECMA, the Mozilla Foundation continued to develop Javascript for the Firefox browser. In 1 June 1997, ECMA-262 Edition 1 was released.

Basic Syntax

Javascript syntax uses two value types. Fixed and variable values. Fixed values are known as literals, numbers are written with or without decimals and strings are text contained within double or single quotes. A variable in Javascript is used to store data, 'var', 'let' or 'const' are used to declare variables. An = is used to assign values to variables, e.g. let x = 6. Operators are used (I.e. +, -, /, *) in expressions to manipulate values and the data within variables. String can also be added to one another to combine them. Comments in Javascript use '//' or '/* */' which ensures that any code in the line onwards are not executed and any codes within the comment slashes will not be executed respectively.

Integrated Development Environment Softwares (IDEs)

Atom is a source code editor used for Javascript development, it is flexible and highly customisable and is free to use. AWS Cloud9 is another IDE that supports Javascript although it requires an AWS account but it comes with an inbuilt terminal that supports npm and Unix commands. Brackets is another IDE that is a favorite among JS developers. It allows live prevew to see how changes affect a project in real-time. Finally, Codeanywhere is an IDE that is cloud-based and allows developers to write, edit and run websites directly from a web browser.

Applications

Javascript is mainly used for creating dynamic and interactive web content like applications and browsers. Likewise, Javascript can be used for game development and building web servers or server applications and is used for presentations or slideshows.
